# python com MySql

Exemplo de inclusão de dados em um banco de dados, usando python.

# O que foi usado neste projeto para configurar a biblioteca mysql no python?
```
poetry init
poetry add mysql-connector-python
```
# Como instalar as bibliotecas na primeira vez que for executar o programa?
```
poetry install
```
# Como executar?
```
poetry run python be_principal.py
```
